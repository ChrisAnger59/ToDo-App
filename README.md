# 📝 ToDo App - Gestionnaire de tâches par colonnes

Application web full-stack de gestion de tâches avec système d'authentification et organisation par colonnes personnalisables.

---

## 🚀 **Technologies utilisées**

### **Backend**
- Node.js
- Express.js
- MongoDB / Mongoose
- JWT (jsonwebtoken)
- Bcrypt

### **Frontend**
- HTML5 / CSS3
- JavaScript Vanilla
- Fetch API

---

## ✨ **Fonctionnalités**

### **Authentification**
- ✅ Inscription avec hashage des mots de passe
- ✅ Connexion sécurisée avec JWT
- ✅ Déconnexion
- ✅ Protection des routes

### **Gestion des colonnes**
- ✅ Créer des colonnes personnalisées
- ✅ Modifier le nom d'une colonne
- ✅ Supprimer une colonne
- ✅ Réorganiser les colonnes

### **Gestion des tâches**
- ✅ Créer des tâches dans une colonne
- ✅ Afficher les tâches par colonne
- ✅ Données liées à l'utilisateur connecté
